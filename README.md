# Trading-App-Fe

## Installation

To set up this project on your local machine, follow these steps:

1. **Clone the Repository**

   ```bash
   git clone https://github.com/West-Coast-Tech/Trading-App-FE.git
   ```
   open the folder

2. **Installaion**

```bash
npm install
```
3. **Run**
```bash
npm run dev
```
